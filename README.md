# IMDb Top 250 Movies (1900–2020)

Power BI dashboard analyzing **IMDb Top 250 movies** from 1900 to 2020.

## Key Stats
- **27 movies** in Top 250 from 1900–2020
- **Average rating**: 8.28
- **Min**: 8.10 | **Max**: 8.70
- **1980s**: Most represented decade

## Features
- Bar chart: Movies by decade
- Table: Rank, title, year, rating
- Cards: Total, avg, min, max ratings
